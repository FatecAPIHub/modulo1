# modulo1
API REST para gerenciamento de pessoas: CRUD completo em Spring Boot 3+, paginação padrão (10 itens por página), filtro por status (ativo/inativo) e integração de logs via Graylog.

# PROD
A inclusão de arquivo `.env` ou outros dados sensíveis no repositório é má prática e está aqui apenas para fins educacionais. Em produção, utilize variáveis de ambiente ou um vault e nunca versione segredos.
